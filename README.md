# Single Axis Ball Screw Servo System

## Overview
This project models and controls a single-axis ball screw drive system using a DC motor, transfer-function modeling, and Simulink-based control design.

- [x] Ball screw modeling
- [x] Transfer function derivation
- [x] Pole analysis
- [x] Open-loop response analysis

- [ ] Disturbance rejection study
