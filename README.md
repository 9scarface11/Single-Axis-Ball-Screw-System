# Single-Axis Ball Screw Drive System

## Overview

This project models and analyzes a single-axis ball screw drive system used for industrial motion-control applications.

A closed-loop PID control architecture was developed using MATLAB and Simulink to regulate actuator position and improve tracking performance.

## Tools

- MATLAB
- Simulink

## Objectives

- Model actuator dynamics
- Implement closed-loop control
- Analyze transient response
- Evaluate position regulation performance

## System Architecture

Reference Position
↓
PID Controller
↓
DC Motor + Ball Screw
↓
Position Output
↑
Feedback Sensor

## Features

- Dynamic system modeling
- PID-based position control
- Closed-loop feedback architecture
- Step response analysis
- Control performance evaluation

## Results

- Stable position regulation
- Reduced settling time
- Improved tracking accuracy
